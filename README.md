# Accdata
Demo Matlab software (*) for pattern recognition using neural networks and support vector machines.



* doc: italian documentation;
* ottimizzazione: source files of optimization technique demo (optim::fmincon);
* sorgente: source files of mean project (k-means etc.);
* svmxor: source files of supp.vec.mach. demo (by hand, no libsvm).

(*) Matlab 2009, Educational licence
